# 🏝️ Zaizai Isle (V1.2.6)

 > 一个可长期维护、可继续生长、可承载个人变化的数字小岛
 > *A digital isle that grows, survives, and reflects personal evolution.*

---

## 1. 产品定位（Reframed）

**Zaizai Isle** 不是一个传统意义上的「个人主页」或「作品集网站」，而是一座 **持续扩建的个人数字小岛**。

*   它是一块 **可以自由生长、长期维护的个人空间**，而非被模板约束的页面。
*   它以 **整体氛围与体验状态** 而非功能清单作为第一驱动力。
*   它是一次 **Vibe Coding 从直觉到精密逻辑的完整落地样本**。

---

## 2. 整体体验目标

> 再做一点，再想一点，惊喜总在发生。
> **Build more, think more, find the spark.**

访客进入 Zaizai Isle 时，应当感受到：
1.  **静止与流动的平衡**：信息安静、有边界，但环境（天气、昼夜）正在经历时间。
2.  **高精度的底座**：系统每一个刻度均经过校准（12 列栅格、物理比例锁定）。
3.  **叙事的层次**：从表层的岛屿隐喻，到深层的系统运行事实，层层自洽。

---

## 3. 核心信息架构（IA）

采用 **12 列 Bento Grid** 布局，每一个卡片都是一个“低干扰的信息单元”：

*   **Identity Anchor**（身份锚点 / 核心卡片）
*   **Climate Logic**（气候节律 / 环境感知）
*   **Node Traces**（节点追踪 / 数据统计）
*   **Social Heart**   （社交组件 / 联络互动）
*   **Built Structures**（已建成结构 / 作品展示）
*   **Tools Dashboard**（工具面板 / AI 能力外显）
*   **Isle Deposits**（岛屿沉积 / 时间留痕）
*   **Core Build**（构建面板 / 实时生长进度）

---

## 4. 核心模块说明

### 🪪 Identity Card｜身份卡片
**定位**：当前状态声明，而非简历。
*   拟人化 IP「鲸头鹳」作为气质锚点，校准全站设计的“冷静感”。
*   中/英双重 Slogan 动态重排，适配不同文化语境下的阅读节奏。
*   通过状态切换（Idle, Working, Alert 等）建立轻量互动。

### 🌤️ Weather Card｜环境感知
**定位**：提供“此刻感”的氛围层。
*   实时天气数据驱动渐变背景与玻璃质感图标。
*   昼夜状态联动，确保页面视觉与物理世界的步调一致。
*   异步加载技术确保环境渲染不阻塞主叙事。

### 📊 Stats Card｜行为与数据
**定位**：记录“发生过什么”，而非展示成就。
*   名片下载次数持久化，记录访客的微弱连接。
*   支持一键生成「社交名片海报」，将数字化身转化为可传播资产。

### 💬 Social Card｜低压力社交
**定位**：冷静、有边界的联络方式。
*   WeChat/Email/GitHub 聚合入口。
*   Like 行为带动效反馈，建立无痛点的轻交互。

### 🛠️ AI Hub & Tools｜工具面板
**定位**：展示工作方式。
*   常用 AI 工作流入口聚合。
*   内置高性能前端工具（如图片压缩），体现“工具即性格”。

### 📂 Works Card｜已建成结构
**定位**：减少展示，强调构建质量。
*   精选项目以“结构”而非“截图”为核心。
*   悬停预览与沉浸式模态框，提供无干扰的阅读体验。

### 📝 Archive｜岛屿沉积
**定位**：时间流逝的痕迹，而非社交动态。
*   弱化实时感，通过留言系统记录“时间的灰尘”。

### 🏗️ Core Build｜构筑面板
**定位**：岛屿生长的实时观测窗口（System Layer）。
*   32 位高分辨率进度条搭配 System Scanner 动态扫描。
*   模块运行状态（ACTIVE / STABLE / FORMING）全景展示。

## 5. 技术架构：精密工程

在 V1.2.6 中，我们引入了 Vercel 等顶级工程实践：
*   **动态渲染拓扑**：利用 `next/dynamic` 实现模块级异步加载，首屏 JS 负载最小化。
*   **渲染帧率优化**：采用 `React.memo` 对装饰性高频组件（云朵、动画）进行物理隔离。
*   **12 列栅格系统**：提供更高精细度的布局控制力。
*   **全量 i18n 集成**：所有状态逻辑均已托管，实现语境级适配。
